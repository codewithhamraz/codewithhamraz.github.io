HTML Kya Hai? Bilkul Aasan Bhasha Me

HTML ka full form hai Hyper Text Markup Language.

1. HTML ka matlab kya hua?

Socho website ek ghar hai. To HTML us ghar ki deewar, darwaza, chhat banata hai. Matlab structure deta hai. Bina HTML ke website khali kagaz jaisi hoti hai.

2. HTML kaise kaam karta hai?

HTML me hum <tag> use karte hain. Tag matlab dabba jisme content likhte hain.

Example Code:
<h1>Main Heading</h1> → Sabse badi heading
<p>Ye paragraph hai</p> → Normal line
<img src="photo.jpg"> → Photo dikh jayegi

3. HTML seekh ke kya fayda?

1. Apni website bana sakte ho
2. Freelancing kaam mil sakta hai
3. Apni site se AdSense ka paisa kama sakte ho
